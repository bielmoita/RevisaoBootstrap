Leiaute


Conteineres

Os conteineres são os elementos de leiaute mais basicos no Bootstrap e são usados para conter, preencher e (as vezes) centralizar o conteudo dentro deles.

Por padrão o bootstrap estabelece os pontos de quebra (media query) para alguns tamanhos de tela/janela.

Exemplos:

.container

.container-sm

.container-md

.container-lg

.container-xl

.container-fluid
