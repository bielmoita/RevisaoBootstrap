Paragrafos

O elemento HTM <p> (o elemento de paragrafo) representa um paragrafo do texto.


Listas

O elemento HTML <ul> (ou elemento de lista desordenada) representa uma lista de itens sem ordem rigida, isto é, uma colação de itens que não trazem uma ordenação numérica e as suas posições.

  • Banana
    • Nanica
    • Prata
  • Abacaxi
  • Graviola

O elemento HTML <ol> (Ou elemento de lista ordenada) representa uma lista de itens ordenados.

  1. Banana
  2. Abacaxi
  3. Graviola

O Elemento HTML <dl> (Ou elemento de lista de definições) engloba uma lista de pares de termos e descrições.

Frutas
Banana, Abacaxi e Graviola.
Legumes
Abobrinha, Cenoura e Batata.

Textos Pré-formatados

O elemento HTML <pre> representa o texto pré formatado que deve ser apresentado exatamente como esta escrito no arquivo HTML.

        *  TEXTO PRÉ-FORMATADO  *
